Instalacao padrao do flutter



1. Abrir o terminal na pasta raiz do disco C;
2. fazer o clone do git; 
   git clone https://github.com/flutter/flutter
3. entrar na pasta clonada
4. abrir o VS Code nesta pasta 
5. mudar para a branch desejada 
6. voltar ao terminal e rodar o comando
   flutter doctor
7. apontar a variavel de ambiente PATH 
        c:/flutte/bin

Obs.  é necessario apontar o flutter nas variaveis de ambiente
